About This Demo

This browser demo showcases the XLS to XLSX Converter's client-side capabilities:

  • Client-Side Conversion: All processing happens in your browser - files are never uploaded to any server
  • Formula Security: Automatically strips potentially dangerous formulas
  • Multiple Formats: Supports XLS, XLSX, CSV, ODS, and JSON input
  • No Installation: Works right in your browser without any software installation

Limitations of Browser Demo

  • Maximum file size is limited by your browser's memory
  • Some complex XLS features may not be fully supported in client-side mode
  • For production use or more complex files, consider using the server-side API
